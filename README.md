## Descripción

Sitio web institucional para **UntrefSchool**, una institución educativa de nivel inicial. El sitio presenta la propuesta pedagógica, equipo docente, propuesta educativa y noticias de la institución.

Proyecto integrador de la materia de **Desarrollo Web** en UNTREF.

## Demo

🔗 **[Ver sitio en vivo](https://cintava.github.io/proyecto-integrador-web-untrefschool/)**

## Tecnologías

- **HTML5** — Estructura semántica
- **CSS3** — Estilos, layout responsive y animaciones

## Estructura del proyecto

```
untrefschool/
├── multimedia/      # Imágenes y recursos gráficos
├── contacto.html    # Página de contacto
├── index.html       # Página principal
├── style.css        # Estilos
└── README.md
```

## Características

- Diseño responsive
- Secciones informativas: quiénes somos, por qué elegirnos, propuesta educativa
- Tarjetas de propuesta educativa con íconos ilustrativos
- Sección de noticias institucionales
- Página de contacto separada
- Footer con información de contacto y redes sociales

## Secciones

| Sección | Contenido |
|---------|-----------|
| Quiénes Somos | Presentación institucional y misión |
| Por Qué Elegirnos | Enfoque pedagógico y equipo docente |
| Propuesta Educativa | Aprendizaje lúdico, arte, educación ambiental, multilingüe, socioemocional |
| Últimas Noticias | Novedades de la institución |
| Contacto | Formulario y datos de contacto |
